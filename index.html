Inspections

Visualization

Dashboards

Mapping

Trend Analysis and Forecasting

Analyze trends based on historical data and make predictions for the future using the built-in machine learning and neural network technology.

Automation

Control switches, motors and actuators using a visual programming environment with drag-and-drop of conditional statements, loops and variables.

Compliance monitoring

Use visual programming to set up rules for anomaly detection and monitor compliance with service-level agreements

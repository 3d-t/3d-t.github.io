3D-T : Spacetime monitoring and control

Integrating the Internet of Things, Machine learning and Digital Twins

Learn moreDownload

IoT devices

Capture data from IoT devices, bluetooth sensors, mobile devices and legacy systems using pull and push data exchange.

Inspections

Use smart forms to automatically or manually collect data from surveys or inspections on mobile devices.

Surveys

Quickly create surveys and tests to assess the opinions and qualifications of your employees, clients or contractors.

Monitoring and control

Use a central dashboard and single application to visualize data and control IoT devices from different sources.

Trend Analysis and Forecasting

Analyze trends based on historical data and make predictions for the future using the build-in machine learning and neural network technology.

Automation

Automate processes and control devices using a visual programming environment with drag-and-drop of conditional statements, loops and variables.

3D visualization

Visualize space-time information in 3D map view with sattelite images, street view, elevation maps or a 3D terrain model.

Parametric Solid Modeling

Use visual programming to create 3D solid models that automatically change shape or color based on sensor values.

Digital Twins

Overlay 3D CAD and BIM models with real-time information to analyze problems and pick on items to retrieve asset information and documents.

HTTP

Instead of entering data via copy and paste into the data window you can also automatically import data via HTTP or HTTPS. To set up an automatic import you first need to specify a data source. When the importers runs the data is collected and you can then process the CSV or JSON by adding entries for mapping objects and keys to topics.

Datasource

Select options Secrets from the sidemenu and create a new entry. Add a field for type and set the value to http. Add a field url and set the value to the http link.

Importer

Create a new importer and under Datasource select the name of the secret that has the details. Select menu option Test to retrieve the data. You can now add mappings for the keys. Note that the fields for latitude and longitude are set to a fixed value that indicate the GPS coordinates of the station. You can automatically run scheduled imports by setting Run to Every day.

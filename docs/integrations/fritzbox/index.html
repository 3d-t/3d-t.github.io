Fritzbox

A router can be used to detect if somebody is present at a location with a WIFI network. The Fritzbox integration connects to the router and retreives a list of devices that are connected.

Usage: Detect if people are present at a location.

Script#

Create a new script with name Fritzbox. The script first connects to the router with the username to request a token called Challenge. This token is combined with the password and encoded to a UTF-16LE string. This is used to request a token and the response is an XML file. The XML is converted to JSON and the session is SID is obtained. In a third call the state of the router is requested via an HTTP request. If the object passed to the get functions is found in the net.devices JSON array the value is set to 1, or else 0.

class Fritzbox {
    init = (context) => {        this.context = context    }
    get = async (object) => {        let connected = await this.objects()        let home = connected.find(item => item.name === object) ? 1: 0        return {object: object, home: home}    }
    objects = async () => {        var sid = await this.sid();        var res = await http.get(this.context.host + '/data.lua?sid=' + sid, "1 minute")        var devices = res.data.net.devices;        return devices    }
    sid = async () => {        var response = await http.get(this.context.host + '/login_sid.lua?username=' + this.context.username, "1 minute");        var json = await xml.toJson(response).then(result => {return(result)}).catch(err=>error(err));        var challenge = json.SessionInfo.Challenge;        var token = encryption.encode(challenge + '-' + this.context.password, "UTF-16LE");        token = encryption.md5(token)        response = await http.get(this.context.host + '/login_sid.lua?username=' + this.context.username + '&response='+challenge+'-'+token, "1 minute");        json = await xml.toJson(response).then(result => {return(result)}).catch(err=>error(err));        return json.SessionInfo.SID    }}

Note that you can test the script before running the importer. To paste the code below in the test field of the script and click Run.

let fritzbox = new Fritzbox()fritzbox.init(await encryption.secret('Fritzbox'));var i = await fritzbox.get('iPhonevanEsther')print(JSON.stringify(i))

Datasource#

Create a new secret with a type set to script and fields for the host, username and password. The host should be the URL to the admin port of the router, such as https://fritzbox.mydomain.com:1443.

Importer#

Create a new importer and set the Datasource to the secret that defines the Fritzbox connection. Add one or more objects for which the script should be called. The field Object should contain the name of the device as it is registed with the router, typically the name of the smartphone.

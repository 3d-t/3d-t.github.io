Time

When the value of a topic is set a new entry in the log is created. This log allows you to track the changes over time, detect anomalies, perform statistical analysis and forecast future values.

Set time#

When you change the value and click on the Update button in a topic screen a new log is created. By default the current date and time are set to the effective date of change. You can override the default date by entering a new date in the Time field. The value can entered with the keyboard or by using the date and time picker. Instead of opening the topic details screen you can also enter the new value and effective date in the search box, for example asset.hvac.bosch.pressure = 2 @ 2012-09-01 10:33 The date and time should be entered in the ISO format. It is also possible to specify the date or time part only.

Timeline#

To graphically view the timeline click on the Chart tab. You can optionally change the period by selecting the From and Until time and click Update. To zoom in to a specific time you can specify an time interval by dragging the pointer. Note that even if the value of a topic has not changed you can update the value to confirm that the value was measured and is displayed in the timeline.
